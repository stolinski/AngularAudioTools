Wave

The wave directive produces a single wave, capable of being modified and turned on or off.

Attributes
Attribute Values Example
freq Any frequency value freq="440.00"
volume Value between 0.0 and 1.0 volume="0.3"
type sine, square, saw or triangle type="sine"
note The note of the wave note="A4"
